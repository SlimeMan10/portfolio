Fitness Tracker

Fitness Tracker, Seattle, WA | HTML, CSS, JavaScript, SQL | Sep 2024 - Present

In 2021, I often found myself frustrated with existing fitness apps that couldn't track the specific intensity techniques I used in my bodybuilding workouts. This led me to envision more specialized tracking solutions. When I began my web development course at UW in Fall 2024, I saw an opportunity to turn this idea into reality.

I started by creating a draft of the user interface, focusing on the features I wished existed in other apps. Using HTML and CSS, I made the website framework. I implemented basic functionality for logging workouts through JavaScript, which helped me understand DOM manipulation and event handling. This project is my first step into web development, showing my ability to apply classroom concepts to solve real-world problems I've experienced. While still in its early stages, working on this project has strengthened my understanding of frontend development fundamentals and motivated me to continue expanding my technical skills.
